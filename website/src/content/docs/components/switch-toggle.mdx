A switch toggle allows users to toggle a boolean state on or off.

## Basic

<Preview src="switch?style=basic" height="150px">

```dart
SwitchToggle(
  value: _value,
  onChanged: (v) => setState(() => _value = v),
)
```

</Preview>

## Disabled

<Preview src="switch?style=disabled" height="150px">

```dart
SwitchToggle(
  value: _value,
  disabled: true,
  onChanged: (v) => setState(() => _value = v),
)
```

</Preview>
