Sliders allow users to make selections from a range of values.

## Default

<Preview src="slider?variant=default" height="200px">

```dart
Slider(
  value: 0.5,
  onChanged: (value) {
    // Handle change
  },
)
```

</Preview>

## Custom Range

<Preview src="slider?variant=range" height="200px">

```dart
Slider(
  value: 50.0,
  min: 0.0,
  max: 100.0,
  onChanged: (value) {
    // Handle change
  },
)
```

</Preview>

## Discrete Divisions

<Preview src="slider?variant=discrete" height="200px">

## Handling Release

<Preview src="slider?variant=handleRelease" height="200px">

```dart
Slider(
  value: 0.2,
  divisions: 5,
  onChanged: (value) {
    // Handle change
  },
  onChangeEnd: (value) {

  },
)
```
</Preview>
