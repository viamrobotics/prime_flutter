Icons are used to represent actions, objects, or concepts.

## Usage

Use the `Icon` widget with `PrimeIcons` constants.

<Preview src="icon?style=example" height="200px">

```dart
Icon(
  PrimeIcons.viamFlutter, 
  size: 24,
)
```

</Preview>

## Gallery

Browse all available icons in the Prime Flutter library. 
Click on an icon to copy its name to the clipboard.

<Preview src="icon?style=gallery" height="1000px">

```dart
// Example of using an icon from the gallery
Icon(PrimeIcons.checkboxIndeterminate)
```

</Preview>
