Filter pills are used to filter content or selections.

## Basic

A basic filter pill with a label.

<Preview src="filter-pill?style=basic" height="150px">

```dart
FilterPill(
  label: Text('Filter'),
  onTap: () {},
)
```

</Preview>

## Selected

A filter pill in a selected state.

<Preview src="filter-pill?style=selected" height="150px">

```dart
FilterPill(
  label: Text('Selected'),

)
```

</Preview>
