Expansion list items are used to show or hide a list of children.

## Standard

The standard style is used within lists.

<Preview src="expansion-list-item?style=standard" height="300px">

```dart
ExpansionListItem(
  initiallyExpanded: true,
  title: Text('Expansion Item'),
  leading: Icon(PrimeIcons.viamFlutter),
  children: [
    Padding(
      padding: EdgeInsets.symmetric(vertical: 8.0),
      child: Text('This is the content hidden inside the expansion item.'),
    ),
  ],
)
```

</Preview>

## Card

The card style has a background and border, making it stand out.

<Preview src="expansion-list-item?style=card" height="300px">

```dart
ExpansionListItem.card(
  initiallyExpanded: true,
  title: Text('Card Expansion'),
  leading: Icon(PrimeIcons.viamFlutter),
  children: [
    Padding(
      padding: EdgeInsets.symmetric(vertical: 8.0),
      child: Text('Content inside a card style expansion item.'),
    ),
  ],
)
```

</Preview>
